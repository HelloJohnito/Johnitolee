EventureBrite

EventureBrite is an event hosting web application inspired by Eventbrite. I built this web application using Ruby on Rails and React.js + Redux. EventureBrite allows users to host events, browse events by categories, purchase tickets, and bookmark their favorite events.

OurMap

OurMap is a mobile application inspired to connect people through picture sharing. Users are able to share their experiences by taking pictures and pinning them by their geolocation. This creates mini albums of photos dropped by different users and stored in locations for other users to view. We developed this app using React Native, Redux, Xcode, Firebase, and Cloudinary.

Meetarii

Meetarii is a web application that utilizes APIs from Meetup and EventBrite. The idea here was to create a site where a user can view events hosted from both sites in one centralized location. Users can view and compare events side by side and see which events they want to attend. Please note: Meetup's API requires the user to turn on CORs.

DodgeyBird

DodgeyBird is web game inspired by the famous Flappy Bird but with its own twist. Instead of going through green pipes, the objective of this game is to dodge missiles while collecting coins! Sounds challenging? Give it a shot!

ShapeBeats

Create unique beats with ShapeBeats! With ShapeBeats, users are able create their own unique beats with the keys 'a' through 'z'. ShapeBeats utilizes paper.js to display the animations and howler.js for the audio.
